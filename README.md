# Stock Trading App Mobile

Project React + Vite yang siap dijalankan dan di-deploy.

## Jalankan lokal
1. npm install
2. npm run dev

## Deploy ke Vercel
1. Ekstrak ZIP.
2. Upload folder project ke GitHub, lalu import repository di Vercel.
3. Vercel akan mendeteksi Vite secara otomatis.
4. Klik Deploy.

Build command: npm run build
Output directory: dist
